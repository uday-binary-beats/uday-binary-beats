2. Add the following workflow file: `.github/workflows/metrics.yml`
   ```yaml

   on:
     schedule: [{cron: "0 0 * * *"}]
     workflow_dispatch:
   jobs:
     github-metrics:
       runs-on: ubuntu-latest
       steps:
         - uses: lowlighter/metrics@latest
           with:

             user: uday-binary-beats
             template: classic
             config_timezone: Asia/Kolkata
   ```
